The Story

Inventor & Founder

Kevin Caracozza has been collecting trading cards for 25 years and running businesses for 26. When he couldn't find a system that actually worked for his collection, he built one.

NEXUS started in a New Jersey lab with a simple idea: what if scanning and selling a card was as easy as scanning a barcode?

Today, NEXUS uses AI-powered recognition to identify cards from over 100 million items, pulls real-time pricing from multiple sources, and lets shop owners list inventory in seconds instead of hours.

101 patent claims filed. Real shops using it daily. This is just the beginning.

The Philosophy

Most card scanning systems are limited by their hardware. Expensive cameras, proprietary sensors, locked-down ecosystems. NEXUS flips the script.

We believe the intelligence should live in the software, not the hardware. NEXUS runs on Raspberry Pi 5s, commodity Edge TPUs, and consumer-grade CZUR scanners. The hardware is replaceable. The software is irreplaceable.

The Architecture

🎯 5-Stage Confidence Waterfall (ACR Pipeline)

Every card goes through a cascading recognition pipeline that exits at ≥95% confidence:

1. Art Recognition

Coral Edge TPU + FAISS vector search against 390K+ card embeddings

2. Enhanced OCR

EasyOCR GPU + Tesseract fallback for set codes & collector numbers

3. Metadata Lookup

Cross-reference against 521K MTG, 19.8K Pokemon, 1.84M sports cards

4. Claude Vision

AI-powered visual analysis for edge cases and rare cards

⚡ Distributed Intelligence

BROCK (Pi 5): Art recognition server with Coral M.2 TPU
SNARF (Pi 5): OCR engine with GPU acceleration
ZULTAN (RTX 3060): FAISS index, metadata API, marketplace server

Each node is specialized. Each node is replaceable. The system is resilient.

What Makes NEXUS Different

🧠 Bulletproof OCR Engine (Patent Claim 2)

Never crashes. Auto-detects Tesseract installation across platforms. Multi-pass validation with 3-tier fallback strategy. Retry logic with exponential backoff. Graceful degradation when things go wrong.

🎨 Coral-Accelerated Art Matching

Runs entirely on a $70 Coral Edge TPU. Extracts 512-dimensional embeddings in milliseconds. FAISS similarity search finds the closest match from 370K+ Magic cards, 19.8K Pokemon cards.

🔗 Multi-TCG Unified API

One API. Three card games. MTG (Scryfall), Pokemon (TCGDex), Sports (TCDB). Normalized schema. Consistent responses. Works with any card type.

🏪 Built-in Marketplace

Scan cards → List for sale → Get paid. All in one system. Stripe checkout. Multi-seller support. Inventory sync from desktop app. The marketplace isn't an add-on—it's part of the core product.

The Vision

NEXUS is software-defined card recognition. The hardware is commodity. The intelligence is distributed. The system is modular.

Want to swap the Coral TPU for a different accelerator? Go ahead.
Want to run OCR on a different Pi? Easy.
Want to replace the CZUR scanner with a flatbed? It's just an image source.

The hardware is a vehicle. The software is the product.
