### Arduino Command Protocol
```
R[000-255] - Red LED value
G[000-255] - Green LED value  
B[000-255] - Blue LED value
W - White LED mode
O - LEDs Off
S[0-180] - Servo position
C - Capture trigger
MOVE X### Y### - Position to coordinates
HOME - Return to zero
SCAN_GRID [cols] [rows] - Batch scan pattern
GET_POS - Report current position
```

## Recognition Algorithms

### Trading Card 4-Corner OCR
```
┌─────────────────────────────┐
│ [TOP-LEFT]     [TOP-RIGHT]  │
│ Card Name      Mana Cost    │
│                             │
│                             │
│                             │
│ [BOT-LEFT]     [BOT-RIGHT]  │
│ Power/Tough    Set + Number │
└─────────────────────────────┘
```
- Fuzzy matching: Levenshtein distance ≤3
- Set symbol CV: 98%+ accuracy
- Processing time: <3 seconds/card

### Stamp Recognition
1. Perforation gauge measurement (edge detection)
2. Watermark detection (transmitted light)
3. Denomination + country OCR
4. Color variant classification (LAB color space)
5. Cancellation type identification

### Coin Recognition
1. Obverse/reverse dual-capture
2. Automatic orientation correction
3. Edge analysis (reeded/lettered/plain)
4. Mint mark detection + OCR
5. Die variety high-res pattern matching
6. Grade estimation (wear analysis)
