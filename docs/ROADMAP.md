## 📅 Release Schedule

### Versioning Strategy
- **Major (X.0.0)**: Breaking changes, major features
- **Minor (2.X.0)**: New features, no breaking changes
- **Patch (2.0.X)**: Bug fixes, security updates

### Release Cadence
- **Patch**: Weekly (as needed)
- **Minor**: Monthly
- **Major**: Quarterly

### Support Policy
- **Current version**: Full support
- **Previous minor**: Security fixes only
- **Older versions**: No support (upgrade required)
